Interactive HSV Color Space Tuner
A utility tool built with OpenCV to determine the optimal HSV (Hue, Saturation, Value) ranges for image masking and object isolation.
This tool is essential for preprocessing in tasks like background removal,green screening, and color-based object tracking.
